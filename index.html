***** CSS layout cheatsheet *****
Display flex changes the block element default to be side by side in a row typically on the main container of dev blocks, this allows flex-direction, align-items, and justify-content. By default flex direction is row, but can be changed to column.
Align-items by defualt is stretched and is especially noticed if the main contianer has a good height. Most used for align-items is center, which alings items vertically. Justify content alings-items horizontal. Default is start of the element, but it can be set to end, and center.
You can also set space-around, or space-between. Align-items and justify-content when combined are perfect for centering vertical and horizontal in parent; However, if flex-direction is set to column then align-items and justify-content roles are reversed.

General Information

Example code:
display: flex;
flex-direction:column;
align-items: center; *** the item is centered on the horizonal axis
justify-content: center; *** aligns the items on the verlical axis

Another code example that is used frequently for logo and nav links is to use space between for containers within the parent, using two elements (see layout#1).

Just remember, the parent is the one controlling all the children, so make sure that you are placing align-items and justify-content with the display-flex in the parent.

Best Practices for CSS

    Principal #1
    Every web page is made up of rows, and each row is a separate idea
    Principal #2
    The backgrounds of rows should not be the same twice in a row (unless it's already visually distinct). Each row should have vertical padding somewhere between 50 to 80 pixels
    Principal #3
    You NEED the content before you lay out a website.
